Se appen her: https://johanhov.github.io/lyd-lab/

Tips: Bruk piltastene til å bytte oktaver på tastaturet (sideveis) og for å bytte presets/lyder (opp og ned).

For de interesserte:

Man kan laste egne lydsett ved å legge til ?presets=[adresse til lydsettfilen] på slutten av adressen til appen, slik som dette: https://johanhov.github.io/lyd-lab/?presets=https://johanhov.github.io/lyd-lab/factory_presets.json

Bytt ut adressen etter presets= med adressen til din egen json-fil med lyder.

Deler man appen med hele denne lenka med elevene, får de alltid dine lyder lastet i tillegg til fabrikk-lydene.

Klikk på Del/Importer-knappen i appen for å få JSON-teksten som representerer lyden din, og kopier denne inn i en tekstfil med filetternavnet .json. NB! Legg merke til at du må ha et ekstra nivå i JSON-fila hvor du legger inn navnet på lyden din. Se fila factory_presets.json som eksempel. Forskjellige lyder skilles med komma.
